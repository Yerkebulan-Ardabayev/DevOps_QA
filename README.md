# DevOps Interview Q&A

273 вопроса и ответа для подготовки к DevOps-собеседованию в одном HTML-файле.

## Возможности

- **273 вопроса** по 13 разделам — от Linux и сетей до Kubernetes и архитектуры
- **Голосовой поиск** — нажми 🎤, скажи термин, и откроется нужный ответ (Web Speech API, `ru-RU`)
- **Умный словарь** — распознаёт русское произношение английских терминов (докер → Docker, кубер → Kubernetes)
- **Глобальный поиск** с подсветкой совпадений
- **Тёмная / светлая тема**
- **Клавиатурная навигация** — `↑↓` листать, `→` вглубь, `←` назад, `/` фокус на поиск
- **Масштаб шрифта** — `T+` / `T-`
- **Без зависимостей** — один HTML-файл, работает оффлайн

## Быстрый старт

### Просто открыть
```
Откройте DevOps_Interview.html в браузере
```

### С голосовым поиском (рекомендуется)
```bash
# Запустить локальный сервер (нужен Python)
cd путь/к/папке
python -m http.server 8080

# Открыть в Chrome
# http://localhost:8080/DevOps_Interview.html
```
> Голосовой поиск требует Chrome/Edge и `localhost` (на `file://` Chrome не сохраняет разрешение микрофона).

## Голосовой поиск

| Скажи | Найдёт |
|-------|--------|
| докер | Что такое Docker? |
| кубер | Kubernetes вопросы |

Микрофон работает в непрерывном режиме — говори термин за термином, не нажимая кнопку каждый раз. Нажми 🎤 ещё раз чтобы выключить.

## Технологии

- Vanilla JS (без фреймворков)
- Web Speech API для голосового поиска
- CSS Custom Properties для тем
- Единый HTML-файл (~1.8 MB)
